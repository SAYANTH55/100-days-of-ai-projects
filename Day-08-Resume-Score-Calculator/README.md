# Resume Score Calculator 🚀

An Advanced Resume Score Calculator built using Python that works like a mini ATS (Applicant Tracking System). This project compares candidate resume skills with job description requirements, calculates a resume match score, detects missing skills, provides improvement suggestions, and generates a visual pie chart report.

The main goal of this project is to help job seekers understand how recruiters and ATS systems filter resumes before shortlisting candidates.

This project includes:
- Resume Skill Extraction
- Job Description Skill Analysis
- Skill Matching Engine
- Missing Skills Detection
- ATS-Friendly Resume Score
- Suggestion Generator for Resume Improvement
- Visual Report using Matplotlib Pie Chart

This project is useful for placement preparation, resume optimization, and understanding real-world hiring systems.
