# Log Analyzer 📊

This project is a simple log analyzer built using Python.
It reads a log file and identifies the number of **ERROR**, **WARNING**, and **INFO** messages.

## Features

* Reads log files
* Detects ERROR messages
* Detects WARNING messages
* Detects INFO messages
* Counts occurrences
* Displays summary

## Technologies Used

* Python
* File Handling

## How to Run

Run the following command:

python log_analyzer.py
