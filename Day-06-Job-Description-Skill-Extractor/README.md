# Job Description Skill Extractor 🧠

This project is a simple skill extractor built using Python.
It analyzes a job description and identifies important technical skills mentioned in the text.

## Features

* Takes job description as input
* Extracts relevant technical skills
* Matches skills using predefined list
* Displays extracted skills

## Technologies Used

* Python
* Basic NLP (string matching)
